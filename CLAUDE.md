# Gabi-diplomi

Converting Macedonian diploma/certificate `.docx` files from 3-page A4
portrait into 2-page A3 landscape (two A4 pages side by side per A3
sheet), while keeping every floating text box live and editable in
Word. See `NOTES.txt` for the detailed, per-document technical log.

## Session start

Before doing any work, read `memory/MEMORY.md` and follow its links to
the relevant memory files. They record which documents are already
converted, which remain, and the validated conversion method — don't
re-derive this from scratch each session.

## Session end

Before ending the session, update `memory/`:
- Update `memory/project_diploma_a3_conversion.md` with any documents
  converted this session and what remains.
- If the working method changed or a new gotcha was discovered, update
  or add a memory file under `memory/` and link it from
  `memory/MEMORY.md`.
- Keep `NOTES.txt` in sync as the detailed technical record; `memory/`
  stays a short index + the non-obvious "why"/"how to apply" notes.
